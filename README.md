# TP React - Gestion de l’état local et interactions utilisateurs avec React

## Introduction
Ce projet est un TP React qui explore la gestion des états et des événements.
Il est composé de plusieurs composants illustrant l'utilisation des hooks
`useState` et `useEffect` ainsi que la gestion des formulaires contrôlés.
